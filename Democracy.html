Democracy

Democracy is een multiplayer web spel die je kan met meerde mensen.

Web App

Dit project is gemaakt met React en Typescript. Dit is gemaakt door Facebook. Het werkt helemaal met node js.

Multiplayer

Dit project is volledig multiplayer. Het werkt met web scokets. De plugin die er voor is gebruikt heet socket.io.

Meesterproef

Dit was gemaakt tijdens mijn stage bij Hulan. Het is gemaakt in 2 weken in opdracht voor Hulan en Sintlucas.
