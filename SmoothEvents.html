Smooth Events

Smooth events is een app voor Android. De app is gemaakt om het makkelijker te maken om met vrienden evenementen te plannen.

Hoe was het gemaakt?

Smooth Events is gemaakt met Xamarin. Dit is een manier om native Android apps te maken in C#. De app gebruikt basis element van Android.

API

De app gebruikt een api die is geschreven in php. Deze api heeft toegang tot een mySQL database. Hierin staan alle gebruikers en evenementen.

Wat heeft de app?

In deze app kan je evenementen aanmaken. Je kan mensen hierbij toevoegen. Als je een evenement met je vrienden hebt gemaakt kan je deze aanpassen. Voor iedereen wordt het dan automatisch geupdate.
