Smooth Events

Smooth events is an app for android. It is created to make it easy to plan events with friends.

How was it made?

Smooth Events was created using Xamarin. It is a fully native android app written in.

API

The app uses an api written in php. This api has access to an mySQL database where all users and events are stored.

What does the app have?

In the app you can create events. Invite people to you events. And when you have created an event and invited the people it will automaticly get images for your event.
