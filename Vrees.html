VRees

Vrees is een vr applicatie die mensen met angstoornissen helpt.

VR

In dit project heeft mijn werk MaMaProducties gebruikt gemaakt van de Oculus Go. Dit is een mobiele versie van de oculus.

Neuro Feedback

In dit project heb ik gewerkt aaan een real time neurofeedback systeem. Hierdoor kan de applicatie aanpassen aan het angstniveau van de patient.

Samen Werking

Dit project is in samenwerking met verschillende bedrijven. Onder andere: de GGze, MaMaProducties, Unit040 en Mind Mansion.
